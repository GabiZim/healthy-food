# Healthy Food Tracker

React/Vite-App fuer GitHub Pages.

Repository-Name: `healthy-food`

Die App speichert Daten lokal im Browser (`localStorage`). Dadurch bleiben Daten beim Neuladen erhalten, aber sie werden nicht zwischen Geraeten synchronisiert.

## GitHub Pages aktivieren

1. Dateien in das Repository `healthy-food` hochladen.
2. In GitHub: Settings -> Pages.
3. Source auf "GitHub Actions" stellen.
4. Bei jedem Commit auf `main` wird die Seite automatisch gebaut.

Die Seite erscheint danach unter:

`https://DEIN-GITHUB-NAME.github.io/healthy-food/`
